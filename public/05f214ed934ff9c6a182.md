`Unity UWP: No Universal Windows Platform module loaded` といわれて色々調べてインストールしまくったのですが，結構大変だったので，メモします．


## Visual Studio 2019でのツールのインストール作業

Visual Studio 2019でUniversal Windows Platform Developmentをクリックし，

![VisualStudio2019_1.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/1771055/83758428-0044-ee19-9678-dc4da022f4cd.png)

「Unityによるゲーム開発」，「C++によるゲーム開発」もついでにクリックしてインストールしておきます．必要かわからないですが，Build設定でC++のなんかのツールが足りないといわれ，「C++によるゲーム開発」もダウンロードして，エラーを解決しました．

![VisualStudio2019_3.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/1771055/8786616d-334c-3deb-d0ab-907866d1b85b.png)


## Unity Hubでのモジュール追加

Universal Windows Platform Build Supportをクリックし，実行する．

![VisualStudio2019_3.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/1771055/8c5e13e8-090f-6496-0f20-6f9a792df917.png)


## Windows (IL2CPP) Target Supportをインストール


![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/1771055/41dee0fb-f213-780c-ace8-c38d494f7e01.png)

`Windows (IL2CPP) Target Support` というリンクをクリックし，パッケージをダウンロード&インストールします．[参考: Where to download the Universal Windows Platform?](https://forum.unity.com/threads/where-to-download-the-universal-windows-platform.504423/)

後はWindows SDKも念のためインストールしておいた方がいいです（なくてもタイトルのエラーは消えますが）．

https://developer.microsoft.com/en-us/windows/downloads/sdk-archive/
