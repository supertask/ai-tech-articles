楕円というのは，結構汎用的なもので，下のように幾何学表現をしたい場合に結構使えます．円にもなるし，単位ベクトルを計算後のY座標にかければ，立体的にな表現もできる．2次元の楕円だけで，楕円体的な表現もできたりします．

![killua_lightning.gif](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/1771055/166e172e-561e-c95a-82eb-cea030922357.gif)

ということで早速「X座標から楕円のY座標を取得する方法」は以下の通りです．

![Ellipse.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/1771055/5a2a144d-6e56-9b78-e198-296dfb8aa59b.png)

これをHLSLで表記した場合は，以下の通りです．

```GLSL
//
// HLSLでの「X座標から楕円のY座標を取得する関数」
//
// 入力: 
//     変数: x
//     固定値: radiusX, radiusY
// 出力:
//     変数: y
float ellipse_y(float x) {
    float y = _RadiusY * sqrt(1 - (x * x) / (_RadiusX * _RadiusX) );
}
```
