## 問題点: Not allowed to access vertex data

Burstコンパイラなどで、SkinnedMeshの読み取り専用のスナップショットを取得する際に、AcquireReadOnlyMeshDataという関数を以下のように取得するのですが（参考は、Keijiro氏の[Smrvfx](https://github.com/keijiro/Smrvfx) ）、

```
var dataArray = Mesh.AcquireReadOnlyMeshData(meshes)
```

そうすると、

```
InvalidOperationException: Not allowed to access vertex data on mesh 'Body' at array index 0 (isReadable is false; Read/Write must be enabled in import settings)
```

というように、MeshのVertexにアクセスできないと言われます。

## 解決法

この問題は、アクセスしようとしているSkinnedMeshやMeshの元となるFBXファイルの「Read/Write」をオンにすることで解決しました！
